Bitácora del proyecto

Esta sección reúne el recorrido de trabajo del sitio: cómo se tomaron decisiones, qué dificultades aparecieron y qué cambios terminaron mejorando el resultado final.

Objetivo

Documentar la evolución del proyecto y dejar registro de las decisiones del equipo.

Alcance

Diseño visual, organización del contenido, estructura HTML y coherencia entre páginas.

Desarrollo del proyecto

1. Primeras decisiones de diseño

Al comienzo se definió que cada integrante tendría una página propia con una identidad visual personalizada, pero todas compartirían una misma estructura base. Para la portada se eligió una composición más neutral, para que funcionara como punto de entrada y no compitiera con las páginas individuales.

2. Dificultades encontradas

La principal dificultad fue evitar que todo se viera demasiado parecido. Había mucha repetición de colores y pesos visuales, por lo que se fue ajustando cada ficha con bordes, fondos y paletas diferentes. También hubo que ordenar mejor las jerarquías tipográficas para que la lectura fuera más clara en móviles.

3. Resoluciones aplicadas

Se simplificaron las estructuras de algunas páginas, se corrigieron enlaces y se unificaron los estilos de encabezados, botones y tarjetas. Además, se revisaron imágenes y nombres de archivos para que el sitio quedara consistente y no dependiera de soluciones improvisadas.

4. Cambios importantes durante la implementación

A medida que avanzó el trabajo, la portada pasó de ser una grilla genérica a una tarjeta de presentación con acceso directo a la bitácora y a los perfiles. También se incorporaron páginas individuales más completas, una identidad visual distinta para cada integrante y una organización más clara del contenido para que el proyecto se sintiera coherente de punta a punta.
