Sobre nuestro equipo

Somos un equipo de estudiantes de la Tecnicatura en Desarrollo de Software enfocados en crear soluciones modernas utilizando HTML, CSS y JavaScript. Este proyecto tiene como objetivo aplicar y consolidar los conocimientos adquiridos en la materia Frontend mediante el desarrollo de un sitio web utilizando HTML, CSS y JavaScript. A través de este trabajo buscamos poner en práctica conceptos fundamentales como la estructura semántica de una página web, el diseño visual mediante estilos y la incorporación de interactividad con JavaScript. Además, el proyecto nos permite trabajar de manera colaborativa, organizando tareas entre los integrantes del equipo y utilizando buenas prácticas de desarrollo. El propósito principal es construir una base sólida en el desarrollo de interfaces web modernas, priorizando la claridad del código, la experiencia del usuario y la correcta organización del proyecto.
